本篇文章粗略记录了本人博客的搭建过程,以供后人参考。

Github Pages

GitHub Pages可以被认为是用户编写的、托管在GitHub上的静态网页。

GitHub Pages本用于介绍托管在GitHub的项目，不过，由于他的空间免费稳定以及有1G的空间大小，并且可以绑定自己的域名，是搭建博客的一个合适选择。

准备工作

安装如下两个搭建博客所必要的软件

  • Node.js .下载LTS版本即可
  • Git. 安装相应位数的标准安装包即可

安装成功后，鼠标点击右键选择Git Bash即可进行相应操作。

添加SSH keys到GitHub

在本地设置好 SSH Key 之后，需要添加到 GitHub上，以完成 SSH 链接的设置。

  • 打开本地 id_rsa.pub 文件（如：C:\Users\用户名.ssh\id_rsa.pub。此文件里面内容为刚才生成的密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。
  • 登陆 GitHub。点击右上角头像中的Settings–SSH and GPG keys — New SSH key。
  • 进入后，Title任取，Key type选择Authentication Key，然后把本地生成的密钥复制到里面Key的文本框中, 点击 Add SSH key 就成功了。

购买和绑定独立域名

国内注册域名平台需实名认证(身份证+手机号)，国外域名注册平台则不需要（只需填写邮箱地址）。你需要权衡个人隐私安全和域名注册价格，选择一个最适合你的域名注册平台。个人比较推荐在Namesilo上购买域名，不仅性价比相比其他网站比较高，且Namesilo提供一年免费的域名隐私保护，可以防止别人通过whois查询获取域名所有者的个人注册信息，可以很好地保护站长隐私。且国内访问无需翻墙，支持支付宝操作。

选择域名后，可解析域名将其连接到你的GitHub静态网页的地址。
