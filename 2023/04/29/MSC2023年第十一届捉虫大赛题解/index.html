签到题、使用指南

电子木鱼

你可通过用鼠标点击2000次获得flag，不过这样做很麻烦，当然用写脚本的方法

先按 F12 查看一下网站的实现代码，查看元素

我们发现这里有一个点击事件，以及提示 ==hit?size=1== ，说明在这种情况下所记下次数加一

所以我们可以python编写代码，直接访问2000次

最后打印出flag

更简单的方法：直接把size后的数改为2000后加到链接上即可得到flag

HIT安全浏览器

打开后提示“您的浏览器不符合规定，请用HIT安全浏览器打开”，明显需要伪造一些东西

可以用BurpSuite抓取数据包并且修改

抓包后修改User-Agent信息（用户代理，存有一些浏览器的信息），将浏览器信息改为HIT安全浏览器，伪造后再send。

之后服务器会返回response，发现状态码是200，同时也可得到flag，问题解决

百密一疏

一串奇妙的代码

flag{MDExMSAwMDExCjAxMTAgMTAwMQowMTEwIDExMDEKMDExMSAwMDAwCjAxMTAgMTEwMAowMTEwIDAxMDEKMDEwMSAxMTExCjAxMTAgMDAxMQowMTEwIDExMTEKMDExMCAwMTAwCjAxMTAgMDEwMQo=}

密码学题。这是base64编码，补0后会得到一个“=”

可以用python脚本解码

解码后得到一长串的01字符串
0111 0011
0110 1001
0110 1101
0111 0000
0110 1100
0110 0101
0101 1111
0110 0011
0110 1111
0110 0100
0110 0101

每一行由二进制转换为十进制，找到对应的ascii码

最后结果为flag{simple_code}

哈工大景色

Misc题目，题目给出这张图片

用Notepad++等文本编辑器打开后，搜索flag即可

屏幕截图

也是图片隐写题，打开后是一个bmp格式图片文件

可以看到图片并没有被完整的展现出来，我们需要对该文件进行小小的修改

网上搜索bmp文件格式，查找bmp文件的格式头

BMP格式的文件从头到尾依次是如下信息：

bmp文件头（bmp file header）：共14字节；
位图信息头（bitmap information）：共40字节；
调色板（color palette）：可选；
位图数据；

bmp文件头（bmp file header）

bmp文件头包含如下信息：

bfType：2字节，文件类型；
bfSize：4字节，文件大小；
bfReserved1：2字节，保留，必须设置为0；
bfReserved2：2字节，保留，必须设置为0；
bfOffBits：4字节，从头到位图数据的偏移；

关键是修改这个图像的高度

打开winHex，这里我将高度修改为04

再用ps软件打开
